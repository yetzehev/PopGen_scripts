SFS computed with ANGSD

SFS built with ANGSD, Sammtools Genotype Lykelihood Model (-GL 1) For 'n' diploid samples, the site frequency spectrum (SFS) is the (2n+1) vector containing the proportion of site carrying 'k'-mutations. This means that the first element in the SFS is the proportion of sites where we don't observe any mutations, The second value is the proportion of sites where we observe 1 mutations. However I assumed an haploid model for Zymoseptoria.

Methods described in Nielsen, 2012

#plot the results
nnorm <- function(x) x/sum(x)

#Load SFS
#Aegilops
sfs.aeg <-scan("../sfs/Zt.aeg.sfs")
sfs.aeg.norm<- nnorm(rbind(sfs.aeg[-1])) #The first value represent the expected number of sites with derived allele frequency equal to 0
  #Plot counts
  barplot(sfs.aeg[-c(1)], names=1:29)

  #Data normalization
  barplot(sfs.aeg.norm, names=1:29)

#Wheat
sfs.wht <-scan("../sfs/Zt.wht.sfs")
sfs.wht.norm<- nnorm(rbind(sfs.wht[-1]))
  #Plot counts
  barplot(sfs.wht[-c(1)], names=1:45)

  #Data normalization
  barplot(sfs.wht.norm, names=1:45)

#expected number of sites with 1:28 derived alleles
sfs.aeg <-sfs.aeg[c(1:29)]
sfs.wht <-sfs.wht[c(1:29)]
res <- rbind(
  AEG=sfs.aeg[-1],
  WHT=sfs.wht[-1]
)
# density instead of expected counts
res <- t(apply(res,1,nnorm))

#plot the none ancestral sites
barplot(res,beside=T,legend=c("AEG","WTH"),names=1:28,main="realSFS non ancestral sites",args.legend = list(x="topright"))
